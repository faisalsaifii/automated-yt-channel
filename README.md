# Automated YouTube Channel

## Run Locally

Create the `config.py` file

```python
IG_USERNAME = "example_username"
IG_PASSWORD = "example_password"
```

Set up a virtual environment

```bash
python -m venv venv
```

Install the required dependencies

```bash
pip install -r requirements.txt
```
